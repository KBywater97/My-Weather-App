## My-Weather-App
A simple weather application built with Python that retrieves real-time weather data from an external API.

## Features
- Fetches live weather data using a REST API
- Displays temperature and weather conditions
- Uses environment variables for secure API key handling

## Technologies Used
  - Python
  - REST API
  - JSON
  - Environment Variables (.env)
 
## Setup
1. Clone the repository
2. Create a .env file in the root directory
3. Add your API key (from: https://openweathermap.org/api)
4. run the app
